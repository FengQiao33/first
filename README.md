# EdgePoster（边缘 AI 海报生成器）

**本项目由阿里云ESA提供加速、计算和保护**

![ESA](docs/esa-badge.png)

## 项目介绍（实用性 / 创意性 / 技术深度）

EdgePoster 是一个“部署即用”的**边缘海报生成器**：输入一句话（活动文案/招聘信息/课程宣传/店铺促销），即可一键生成多套适配社媒的海报版式（9:16 / 1:1 / 16:9），并支持导出 PNG。

- **实用性（应用价值）**：自媒体、运营、店主、学生社团都能直接用；不需要安装软件，打开网页即可生成并导出。
- **创意性（创意卓越）**：内置多套流行“赛博/极简/霓虹/杂志风”模板，支持动态背景与字体排版，生成效果适合传播。
- **技术深度（技术探索）**：为边缘场景准备了 `edge` API 示例（可选接入大模型、可加缓存/限流/边缘存储），前端导出采用 Canvas/WebGL 方案，静态资源可被 Pages 边缘加速缓存。

## 本地开发

### 1) 安装依赖

```bash
npm i
```

### 2) 启动开发服务器

```bash
npm run dev
```

### 3) 构建

```bash
npm run build
```

## 部署到阿里云 ESA Pages（参考“将 Github 项目部署至 Pages”）

### 构建配置建议

- **Build Command**：`npm run build`
- **Output Directory**：`dist`

> 部署后得到一个可访问 URL，把它填到 `SUBMISSION.txt` 里提交即可。
